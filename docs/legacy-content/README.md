# Legacy-Content (JSON)

Diese JSON-Dateien waren das ursprüngliche Inhaltsformat der KI-Schulungsplattform.

**Sie werden von der Anwendung nicht mehr geladen.** Die Schulungsinhalte
(Lektionen, Quiz und Nachschlagewerk) leben jetzt ausschließlich in den
TSX-Kapiteldateien unter `schulungsplattform/frontend/src/chapters/`.

Die Dateien bleiben hier nur als Referenz erhalten – z. B. um beim Übertragen
weiterer Inhalte in TSX nachzuschlagen. Sie können gelöscht werden, sobald sie
nicht mehr benötigt werden.

| Datei | Kapitel |
|---|---|
| `01-llm-grundlagen.json` | Wie LLMs funktionieren |
| `02-ki-agenten.json` | KI-Agenten |
| `03-was-ist-agent.json` | Was KI-Agenten sind |
| `03-agenten-bauen.json` | Eigene Agenten erstellen |
| `04-workflows-deployment.json` | Workflows & Deployment |
